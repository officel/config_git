# config_git

my git config on $XDG_CONFIG_HOME/git/

## usage

```bash
cd ~/.config  # $XDG_CONFIG_HOME or $HOME/.config
git clone https://github.com/officel/config_git.git ./git
```

- install id_rsa (manual)
- fix git config remote.origin.url after clone
